Summary



This project analyzes a set of historical NBA game data (through the conclusion of the 2015 season) obtained from fivethirtyeight. The data was subsequently filtered for records pertaining only to the historical performance of the Chicago Bulls. Various analyses were conducted on this filtered data, and the resulting charts, trends, observations and a final data table are available on this site. A chart comparisons page can be accessed via the image link above, or use the navigation bar located on the top of the page, or the preview table located on the right with direct links to each of the individual charts and related analysis.
